# Application Dependencies

This document lists all core dependencies used in both frontend and backend.

---

## Frontend Dependencies (Angular)
- **Node.js** (v14 recommended)
- **Angular CLI**
- **TypeScript** (compatible with project version)
- @angular/* libraries
- RxJS
- Zone.js
- Bootstrap / styling libs (if any)

---

## Backend Dependencies (Node.js API)
- **Node.js / npm**
- **Express** – REST API framework
- **Sequelize** – ORM for PostgreSQL
- **pg / pg-hstore** – PostgreSQL adapter
- **dotenv** – Environment variable management
- **AWS SDK** – S3 media upload/download
- **CORS** – Enabling cross-origin requests
- **bcrypt / JWT** – Authentication (if used)

---

## System Dependencies
- AWS CLI
- Elastic Beanstalk CLI
- TypeScript (if used during build)
- ts-node / ts-node-dev (local development)

---

## Build & Deployment Tools
- CircleCI (CI/CD pipeline)
- Elastic Beanstalk deployment environment
